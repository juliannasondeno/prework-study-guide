HTML

  • The head element is a container for metadata which is not visible to the user when they visit the webpage.
  • The meta element defines metadata or information about the webpage that is used by the browser.
  • The title element defines the title for the webpage and is crucial for Search Engine Optimization.
  • The body element contains the visible content shown to the user when they visit the webpage.
  • The h1, h2, h3, h4, h5, and h6 elements represent the level of heading, from largest to smallest, a given text block represents.
  • The p element represents a paragraph or block of text.
  • The ul, ol, and li elements represent unordered lists, ordered lists, and list items.
  • The img element contains information about images that are displayed on the webpage.
  • The src attribute defines the location of an image or link.
  • The alt attribute contains a text string that describes the appearance and functionality of the image.
  • The br element creates a line of empty space, or line break, between two blocks of content.
  • The a element, or anchor, creates links to the same webpage or other webpages.
  • The link element creates a connection to the external file.
  • The rel attribute specifies the relationship between the current document and the linked document resource.
  • The href attribute specifies the location, or URL, of the external resource.
  • The section element contains a class attributes which adds styles to certain elements by assigning the rule with a class selector in CSS.
  • The script element is used to embed executable code and commonly refers to JavaScript code.

CSS

  • Inline styles apply to a specific HTML tag, using a style attribute to style a particular page element only.
  • Internal style sheets contain CSS rules for the webpage in the head section of the HTML file.
  • External style sheets use a separate CSS file linked within the head section of the HTML file.
  • A margin indicates how much space there is around the outside of an element.
  • A padding indicates how much space there is around the content inside an element.
  • A border surrounds the padding and its content and is on the inside of the margin.
  • Declarations contain the property applied and the value of the property.
  • The * symbol applies a rule to all the elements visible on the page.
  • A comma combines two selectors into a single CSS rule, so the declarations affect both selectors.
  • The width, height, background-color, and color can be adjusted within the header and footer selectors.
  • Semantic references and hexadecimal code, or hex #, both refer to and assign colors.
  • The text-align rule is a positioning property that can adjust headings, paragraphs, and lists.
  • The img selector assigns property values to any image element.
  • The display property assigned with the block value assigns each image to start on a new line.
  • The height, width, margin-left, and margin-right can be adjusted within the img selector.
  • The padding and font size rule can adjust paragraphs and lists.
  • The box-shadow property adds shadow effects around an elements frame and defines the width of the shadow on the x and y axis.
  • A period preceding a word designates a class selector. Any HTML element assigned to the class will gain those CSS properties.

Git

  • git status: checks what branch you are currently on and shows modified files in working directory staged for next commit
  • git branch: lists all branches and the current branch will be highlighted
  • git branch branch-name: creates a new branch at the current commit
  • git merge branch-name: merges the specified branch's history into the current branch
  • git checkout branch-name: switches to the branch named
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add file-name: adds the modifications of the file named in the current working branch to the staging area
  • git add -A: adds all modifications in the current working branch to the staging area
  • git reset file-name: unstage a file while retaining the changes in working directory
  • git commit -m "descriptive message": commits or saves staged changes with a message
  • git pull origin branch-name: receives and merges any commits from the tracking remote branch
  • git push origin branch-name: sends local branch commits to the remote repository branch
  • git config --global user.name "name": sets a name that is identifiable for credit when version history is viewed
  • git config --global user.email "email": sets an email address that will be associated with each history marker

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • A string variable is a sequence of characters used to represent text contained within quotation marks.
  • A Boolean variable is a logical data type that can only have the values true or false.
  • An array stores multiple values under a single variable name by attaching a number, or index, to each item starting at 0.
  • An array can contain different types of data such as string variables, number variables, and Boolean variables.
  • A semicolon is used to end a statement and curly brackets surround code that will be executed.
  • The assignment operator, = , assigns a value to a variable.
  • The strictly equal operator, === , checks to see if two values are equal and returns a Boolean result.
  • The console.log method logs the variable to the web console in the browser.
  • Control flow is the order in which a computer executes code in a script.
  • An if statement is a type of conditional that executes a block of code if the condition is true.
  • An else if statement allows for more than one condition.
  • A for loop is used to repeat a single block of code by using the pattern of index numbers to perform a task on all items in an array.
  • The let keyword declares a variable and allows the value to be reassigned.
  • A for loop begins with the for keyword, contains a three part conditional statement, and a code block that defines what will be repeated.
  • The three part conditional statement in a for loop contains a starting point, a condition, and a pattern.
  • The starting point declares a variable and its value; to start at the beginning of the array, set the value to zero.
  • The condition uses the length property, variablename.length, to state that the variable must be less than the length of the array.
  • The pattern allows the array to repeat over each item; x+1 or x++ increases the variable by one .
  • A function is a reusable block of code that contains instructions on how to perform a specific task.
  • A function must be declared and called in order to be executed.
  • To declare a function, create a function declaration and a code block.
  • A function declaration contains the function keyword and the function name followed by parentheses.
  • A code block is surrounded by curly brackets and lists out the instructions for the function using console.log statements.
  • To call a function, list the name of the function where the code will be executed.
  • Math.floor() and Math.random() are properties that can be used together to generate a random number.
